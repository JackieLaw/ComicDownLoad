# ComicDownLoad
以前一时兴起写的一个WinForm漫画下载器，可以下载漫画，还有一些功能有待完善，不定期更新,界面库使用CSkinUI 和 devexpress,<br>
需要安装devexpress 15.0才能编译。<br>

尚未实现的功能：<br>
1.漫画人图片源解析<br>
2.书签功能的完善<br>

## 编译要求:<br>
VS2017<br>
devexpress 15.0<br>
